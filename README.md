## 🚀 Quick Start

```bash
git clone https://github.com/your-username/nodejs-real-world-app.git
cd nodejs-real-world-app
npm install
````

**Setup Environment Variables**

```env
PORT=5000
MONGO_URI=your_mongodb_connection_string
JWT_SECRET=your_jwt_secret
```

**Run App**

```bash
npm run dev
```

> 🌟 Open `http://localhost:5000` and watch your backend in action!
